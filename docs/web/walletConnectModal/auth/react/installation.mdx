## Implementation

Start by adding `WalletConnectModalAuth` component at the root of your app.

```tsx
import { WalletConnectModalAuth } from '@walletconnect/modal-auth-react'

function App() {
  return (
    <>
      <HomePage />

      <WalletConnectModalAuth
        projectId="YOUR_PROJECT_ID"
        metadata={{
          name: 'My Dapp',
          description: 'My Dapp Description',
          url: 'mydapp.com',
          icons: ['https://mydapp-logo.png']
        }}
      />
    </>
  )
}
```

## Use sign in hook

Set up and use signIn method to open the modal and start user authentication.

```tsx
import { useSignIn } from '@walletconnect/modal-auth-react'

function HomePage() {
  const { signIn, data, error, loading } = useSignIn({ statement: 'Connect to my dapp' })

  async function onSignIn() {
    const data = await signIn()
    console.info(data)
  }

  return <button onClick={onSignIn}>Sign in with wallet</button>
}
```
