# Hooks

## useSignIn

Hook to create sign in method with given options. Returns authentication data.

### Example

```tsx
import { useSignIn } from '@walletconnect/modal-auth-react'

const { signIn, data, error, loading } = useSignIn({
  statement: 'Hello from my dapp',
  chainId: 'eip155:1',
  aud: 'https://mydapp.com/authenticate',
  domain: 'www.mydapp.com'
})

signIn()
```
