## Add packages (NPM)

```bash npm2yarn
npm install @walletconnect/modal-auth-html
```

## Add packages (CDN)

WalletConnectModalAuth uses [unpkg](https://www.unpkg.com/) as its official CDN provider.
Instead of using the npm package, you can simply import it from their CDN and use it in the same way.

See full [CDN example](https://github.com/WalletConnect/web3modal-examples/tree/main/walletconnect-modal-auth-html-cdn) for more info.

```html
<script type="module">
  import { WalletConnectModalAuth } from 'https://unpkg.com/@walletconnect/modal-auth-html'
</script>
```

## Implementation

Start by initializing `WalletConnectModalAuth` in your app.

```js
import { WalletConnectModalAuth } from '@walletconnect/modal-auth-html'

const modal = new WalletConnectModalAuth({
  projectId,
  metadata: {
    name: 'My Dapp',
    description: 'My Dapp Description',
    url: 'mydapp.com',
    icons: ['https://mydapp-logo.png']
  }
})
```

## Call sign in method

Call signIn method to open the modal and start user authentication.

```html
<button id="signin-button">Sign in with Wallet</button>
```

```js
async function onSignIn() {
  const data = await modal.signIn({ statement: 'Connect to my dapp' })
  console.info(data)
}

document.getElementById('signin-button').addEventListener('click', onSignIn)
```
