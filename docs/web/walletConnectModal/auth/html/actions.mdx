# Actions

## signIn

Action to create sign in request with given options. Returns authentication data.

### Example

```js
const data = await modal.signIn({
  statement: 'Hello from my dapp',
  chainId: 'eip155:1',
  aud: 'https://mydapp.com/authenticate',
  domain: 'www.mydapp.com'
})
```

### Reference

```ts
signIn: (options?: SignInOptions) => Promise<SignInData>
```
