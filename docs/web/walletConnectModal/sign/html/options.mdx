## WalletConnectModalSign options

Following options can be passed to `WalletConnectModalSign` constructor as `modalOptions` argument, to customize it i.e.

```js
const modal = new WalletConnectModalSign({
  modalOptions: {
    themeMode: 'dark'
  }
})
```
