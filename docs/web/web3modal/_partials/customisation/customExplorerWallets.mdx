You can manage wallets fetched from explorer via [`explorerRecommendedWalletIds`](./options#explorerrecommendedwalletids-optional) and [`explorerExcludedWalletIds`](./options#explorerexcludedwalletids-optional) options to prioritize, include or exclude them.
To fully disable explorer wallets, use [`enableExplorer`](./options#enableexplorer-optional) option.

You can get all wallet id's from [WalletConnect explorer](https://walletconnect.com/explorer?type=wallet) (click copy icon on chosen wallets).
